# 리덕스(Redux)

> 리덕스를 사용하는 이유

리덕스는 여러 컴포넌트가 동일한 상태를 보고 있을 때 굉장히 유용합니다! 또, 데이터를 관리하는 로직을 컴포넌트에서 빼면, 컴포넌트는 정말 뷰만 관리할 수 있잖아요! 코드가 깔끔해질테니, 유지보수에도 아주 좋겠죠. 🙂

>리덕스의 3가지 특징

- store는 1개만 쓴다!
- store의 state(데이터)는 오직 action으로만 변경할 수 있다!
- 어떤 요청이 와도 리듀서는 같은 동작을 해야한다!


### 리엑트에서 redux 설치하는 방법
<pre>
yarn add redux

yarn add react-redux
</pre>
